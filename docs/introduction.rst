Introduction
============

.. include:: includes/all.rst

Elastic `Beats <https://www.elastic.co/products/beats/>`_ are lightweight data
shippers for transferring log messages, application and host metrics and much
more into a log parser connected to Elasticsearch.

The ``debops-contrib.elasticbeats`` role installs and configures various types
of ``beats`` services.


Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.0.0``. To install it, run:

.. code-block:: console

   ansible-galaxy install debops-contrib.elasticbeats
